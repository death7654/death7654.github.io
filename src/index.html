Chrultrabook Tools

A tool that adds back lost functionality and adds new features to all chromebooks when they install an alternate operating system. Initially, it was created in Electron, but due to its large bundle size the application was remade in Tauri. Some added features include diagnostics, fan controls, and Activity Light controls.
